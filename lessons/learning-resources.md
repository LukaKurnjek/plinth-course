# Learning resources 

To further learn about Cardano smart contract languages you can read the following Mastering Cardano chapters: 
* [Writing smart contracts](https://github.com/input-output-hk/mastering-cardano/blob/main/chapters/chapter-08-writing-smart-contracts-plutusv3.adoc) that includes sections: 
  * [Smart contract programming languages](https://github.com/input-output-hk/mastering-cardano/blob/main/chapters/chapter-08-writing-smart-contracts-plutusv3.adoc#smart-contract-programming-languages) which gives and overview of smart contract language types
  * [Plutus in comparison to Bitcoin Script and Solidity](https://github.com/input-output-hk/mastering-cardano/blob/main/chapters/chapter-08-writing-smart-contracts-plutusv3.adoc#chapter8-plutus-in-comparison-to-bitcoin-script-and-solidity)
  * [Plinth in comparison to Aiken](https://github.com/input-output-hk/mastering-cardano/blob/main/chapters/chapter-08-writing-smart-contracts-plutusv3.adoc#plinth-in-comparison-to-aiken)
  * [Smart contract security](https://github.com/input-output-hk/mastering-cardano/blob/main/chapters/chapter-08-writing-smart-contracts-plutusv3.adoc#smart-contract-security) that provides an overview of EUTXO and Plutus security features 
* [Decentralized applications](https://github.com/input-output-hk/mastering-cardano/blob/main/chapters/chapter-09-decentralized-applications.adoc)

The [Cardano developers](https://developers.cardano.org/) page provides useful guides and explanations for Cardano in general. 

To learn more about the Aiken smart contract language see: 
* *Aiken module* of the [Cardano education program](https://github.com/iohkedu/cardano-education-program/tree/main)
* Official [Aiken docs](https://aiken-lang.org/installation-instructions) that cover: 
  * [EUTxO crash course](https://aiken-lang.org/fundamentals/eutxo)
  * [Aiken language tour](https://aiken-lang.org/language-tour)
  * [Examples](https://aiken-lang.org/example--hello-world/basics)
  * [Common Design Patterns](https://aiken-lang.org/fundamentals/common-design-patterns)
  * [What I wish I knew](https://aiken-lang.org/fundamentals/what-i-wish-i-knew)
* [Awsome Aiken](https://github.com/aiken-lang/awesome-aiken) repository
